# Salon-Azzurro
http://www.azzurrokeramika.me/

Salon AZZURRO is a ceramics and equipment store which different types of equipment from world’s leading manufactures. Homepage is made using Bootstrap 4 carousel (autoplay). Project is animated with GSAP and Animate.css animtations triggered by Intersection Observer API. Each brand has its own card and they are put together on slider which is made using Owl Carousel.

Date of production: September 2021
